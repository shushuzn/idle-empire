## 🎯 游戏技巧

### 前期（0-1M 金币）
- 优先升级金矿和伐木场
- 尽快解锁第一个 Boss
- 保存金币购买关键升级

### 中期（1M-1B 金币）
- 平衡发展所有建筑
- 收集神器获得被动加成
- 完成成就获得碎片奖励

### 后期（1B+ 金币）
- 准备转生，获得灵魂宝石
- 在灵魂商店购买永久升级
- 挑战高难度 Boss

### 转生策略
- 首次转生：1B 金币（获得 1-5 灵魂宝石）
- 第二次转生：10B 金币（获得 2-10 灵魂宝石）
- 优先购买：永恒财富 > 永恒建造 > 永恒之力
