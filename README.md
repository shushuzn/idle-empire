## 🎮 核心功能

### 建筑系统 (12种)
```
⛏️ 金矿 → 🪓 伐木场 → 🌾 农场 → 🏭 工厂 → 🏦 银行 → 🏰 城堡 → 🛕 神殿 → 🚀 太空站 → ⚛️ 量子实验室 → ☀️ 戴森环 → ⏰ 时间机器 → 🌀 多元宇宙门户
```

### Boss系统 (7个)
```
👺 哥布林 → 🧌 巨魔 → 🐉 巨龙 → 👿 恶魔 → 👽 泰坦 → 🌌 虚空领主 → 🤖 机械主宰
```

### 升级系统 (15种)
- 💪 点击强化、🏗️ 建筑加速、👹 Boss伤害、🌀 多元宇宙强化等

### 成就系统 (18个)
- 覆盖建筑、Boss、累计收益、点击次数等多维度

### 收藏品系统 (12个)
- 不同稀有度的收藏品，提供各种永久加成

### 里程碑系统 (18个)
- 长期目标达成后获得永久奖励

### 随机事件系统
- 各种随机触发的事件，提供额外奖励

### 每日任务系统
- 每日更新的任务，完成获得奖励

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `1` | 建筑标签 |
| `2` | 升级标签 |
| `3` | Boss标签 |
| `4` | 成就标签 |
| `5` | 统计标签 |
| `Space` | 点击金币区域 |
| `S` | 保存游戏 |
| `Q` | 购买模式 x1 |
| `W` | 购买模式 x10 |
| `E` | 购买模式 MAX |
| `R` | 王朝重铸 |
